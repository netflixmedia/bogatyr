#  HowTo                  #
###########################

1) Ensure that Java 1.5 is installed on your machine (http://java.sun.com)


2) Simply copy the five folders 

- bin
- cfg
- lib
- res
- src

to your project workspace or import it in your IDE.


3) optional: edit the cfg/ch/orwell/helloworld/standard.properties file and enter the correct path settings


4) If you use an IDE, go to run and select as main method ch.orwell.bogatyr.controller.Runner. 
As parameter ch.orwell.helloworld.HelloWorld /cfg/ch/orwell/helloworld/standard.properties. Now run the application.

Without IDE, you must first compile your application with javac and enter this on your command prompt:

java -jar /lib/bogatyr_0.35.jar ch.orwell.helloworld.HelloWorld /cfg/ch/orwell/helloworld/standard.properties 


5) enjoy :-)


6) if not, check your classpath settings and your java installation...
