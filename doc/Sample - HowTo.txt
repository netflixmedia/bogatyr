+-------------+
| Preparation |
+-------------+
- Ensure that Java 1.5 or higher is installed on your machine (http://java.sun.com)

- Simply copy the folders to your project workspace or import it in your IDE: 

cfg
lib
res
src
(and optional bin)


+------------+
| HelloWorld |
+------------+
optional: edit the cfg/ch/sisprocom/sample/helloworld/standard.properties file and enter the correct settings

If you use an IDE, go to run and select as main method ch.sisprocom.bogatyr.controller.Runner. 
As parameter ch.sisprocom.bogatyr.sample.helloworld.HelloWorld /cfg/ch/sisprocom/bogatyr/sample/helloworld/standard.properties. Now run the application.


Without IDE, you must first compile your application with javac and enter this on your command prompt:

java -jar /lib/bogatyr_0.35.jar ch.sisprocom.bogatyr.sample.helloworld.HelloWorld /cfg/ch/sisprocom/bogatyr/sample/helloworld/standard.properties


+-------------+
| FileManager |
+-------------+
optional: edit the cfg/ch/sisprocom/sample/filemanager/standard.properties file and enter the correct settings

If you use an IDE, go to run and select as main method ch.sisprocom.bogatyr.controller.Runner. 
As parameter ch.sisprocom.bogatyr.sample.filemanager.FileManager /cfg/ch/sisprocom/bogatyr/sample/filemanager/standard.properties. Now run the application.


Without IDE, you must first compile your application with javac and enter this on your command prompt:

java -jar /lib/bogatyr_0.35.jar ch.sisprocom.bogatyr.sample.filemanager.FileManager /cfg/ch/sisprocom/bogatyr/sample/filemanager/standard.properties
